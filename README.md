# 京东手机购物车页面
页面是由内部集成AngularJS和cordova的ionic1.X版本来开发的<br><br>
主要功能包括添加购物车、购物车总价格及总数量自动计算、<br><br>
购物车商品全部选中及单个选中计算价格数量或者编辑删除等<br><br>
